IQmodelo

Statistical Software for Image Quality Assessment with Model Observers

Download this project as a .zip file Download this project as a tar.gz file
The IQmodelo package consists of statistical software for task-based image quality assessments using mathematical model observers. All software is written for MATLAB. IQmodelo was developed at the U.S. Food and Drug Administration, Center for Devices and Radiological Health. Please see the manual for the contents and further information.
